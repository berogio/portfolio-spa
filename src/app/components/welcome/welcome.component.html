About this Portfolio

Thank you for visiting my portfolio! This platform serves as a showcase for my skills and projects. It's meticulously crafted with Angular, utilizing MongoDB for efficient data storage. The backend infrastructure is developed using NestJS and seamlessly hosted on AWS EC2 instances. S3 plays a pivotal role in managing static files, and the Angular frontend is also deployed on its dedicated EC2 instance. To ensure optimal performance and scalability, Nginx and Docker are integral components of the deployment architecture.
